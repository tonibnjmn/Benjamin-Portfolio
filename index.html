PROJECTS

ASSET NEST

Description

Asset Nest is a comprehensive centralized asset management system designed to effectively track and log all assets used within a company. This system aims to streamline asset management processes, enhance accountability, and optimize resource allocation. Its key benefits include enhanced efficiency by automating asset tracking and management, cost savings through the identification and elimination of unnecessary expenditures, improved accuracy by reducing human error, and better decision-making through detailed asset reporting.

Role

As a Full-Stack Developer for the Asset Management System, I played a key role in both front-end and back-end development, ensuring seamless integration and functionality. My responsibilities extended across the entire development lifecycle, including planning, design, testing, and deployment. Additionally, I was in charge of creating the system's technical documentation, which included a comprehensive user guide to assist end-users. The project leveraged modern technologies such as Node.js, React, JavaScript, and Laravel to deliver a robust and user-friendly solution.

Features

The system ensures secure login for authorized users and enables seamless asset management, including data entry, editing, viewing, printing, and hiding. Each asset is assigned an automatically generated tag based on the company, location, category, and ID. The mobile phone management module enhances resource tracking by logging details such as recipient, date, and purpose of transactions, ensuring efficient oversight of all company assets.

ZERINE: A Smartwatch-based Epilepsy Alert System using Non-EEG Physiological data and GPS coordinates via SMS notification

Description

Epilepsy is a neurological disorder marked by recurring seizures due to irregular brain activity, which can be unpredictable and vary in severity, making diagnosis and management challenging. Doctors often rely on reports from patients, caregivers, and witnesses to understand seizure types, triggers, frequency, and their impact on daily life. To address these challenges, the study proposes using smartwatches to monitor metrics like heart rate and movement, allowing communication with healthcare providers and family members. Researchers developed a real-time seizure detection and response system utilizing smartwatch features such as heart rate monitoring and GPS to detect seizures and send location data to designated contacts via SMS. This system aims to enable timely responses to prevent injuries, while also generating detailed records of seizures to help improve epilepsy management.

Role

I served as the UI Designer for Zerine's mobile application, creating a seamless and user-friendly interface. Alongside my design responsibilities, I prepared detailed technical documentation and managed the Alpha and Beta Testing phases, ensuring the application's functionality and reliability for deployment

Features

The system allows users to record detailed seizure information, set personalized BPM triggers for alerts, and store emergency contacts for quick access during an attack. It features an Epilepsy Facts Slideshow for awareness and enables users to generate a PDF report of seizure data for healthcare providers.

DROPS OF HOPE

Description

Drops of Hope is an information management system designed for Blood Donation partnerships, collaborating with medical clinics, hospitals, and other healthcare organizations to streamline and enhance blood donation drives. It offers key benefits such as significantly reducing the time spent managing data, providing robust tools for data analysis, promoting a disciplined approach to data management, transforming disparate information into valuable resources, and improving the quality and consistency of information. Drops of Hope ensures a more efficient, effective, and safer blood donation process for all involved.

Role

I was assigned on the Account Management Module , where I was responsible for creating, updating, activating, deactivating accounts, and managing password changes. This involved performing CRUD operations and implementing authentication features, enhancing my skills in user account management and security.

Features

Drops of Hope features several key modules for efficient blood donation and blood bank management:

  1. Donation Site Information Module: Add or update site details like location, manager, office hours, and days.
  2. Blood Extraction Site Management Module: Manage blood bank affiliations, including location, manager, contact details, blood types, and quantities.
  3. Blood Bank Affiliation Management Module: Oversee blood stocks by adding, releasing, and reserving quantities for specific locations and recipients.
  4. Blood Bank Inventory Management Module: Handle blood type quantities, manage locations, and track recipients.
  5. Blood Extraction Appointment Module: Create, update, and cancel appointments.
